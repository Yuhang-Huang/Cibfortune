# Qwen3-VL-8B-Instruct 使用指南

## 🚀 快速开始

### 1. 测试本地模型
```bash
python test_local_model.py
```

### 2. 快速演示
```bash
python quick_demo.py
```

### 3. 基础功能
```bash
python qwen3_vl_basic_usage.py
```

### 4. 高级功能
```bash
python qwen3_vl_advanced_usage.py
```

## 📁 文件说明

### 核心文件
- `test_local_model.py` - 测试本地模型是否正常
- `quick_demo.py` - 快速演示，创建测试图像
- `qwen3_vl_basic_usage.py` - 基础功能示例
- `qwen3_vl_advanced_usage.py` - 高级功能（OCR、空间感知、视觉编程等）

### 辅助文件
- `quick_start.py` - 完整安装脚本（首次使用）
- `install_dependencies.py` - 依赖安装
- `test_installation.py` - 安装测试
- `requirements.txt` - 依赖列表

## ✨ 主要功能

- **图像理解** - 描述、分析图像内容
- **OCR识别** - 提取图像中的文字
- **空间感知** - 分析物体位置关系
- **视觉编程** - 从图像生成HTML/CSS/JS代码
- **多图像对比** - 比较多张图像的异同
- **多轮对话** - 基于图像的连续对话

## ⚠️ 注意事项

- 模型路径：`/data/storage1/wulin/models/qwen3-vl-8b-instruct`
- 需要至少16GB内存
- 支持GPU加速（自动检测）
- 首次运行可能需要几分钟加载模型
